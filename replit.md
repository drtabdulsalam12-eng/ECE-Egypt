# تطبيق ECE-Egypt للتوظيف

## Overview
تطبيق ويب احترافي باللغة العربية يهدف إلى ربط الباحثين عن عمل بالشركات بكفاءة. يسهل المشروع عملية التسجيل والبحث عن وظائف للباحثين عن عمل، وإدارة طلبات التوظيف للشركات. يطمح المشروع لتوفير منصة شاملة ومتكاملة لسوق العمل المصري، مع التركيز على تجربة مستخدم سلسة وموثوقة.

## User Preferences
أفضل الشرح باللغة العربية الفصحى.
أفضل منهجية عمل تتبع التطوير التكراري (Iterative Development) مع تحديثات منتظمة.
أرجو أن يتم سؤالي قبل إجراء أي تغييرات هيكلية كبيرة أو إضافة ميزات جديدة.
لا تقوم بإجراء تغييرات على الملفات الموجودة في مجلد `public/js/` إلا بعد موافقتي الصريحة.
عند طلب ميزة جديدة، أود الحصول على شرح تفصيلي لكيفية تنفيذها والملفات المتأثرة قبل البدء.

## System Architecture
يعتمد التطبيق على بنية متكاملة تتكون من واجهة خلفية (Backend) قوية مبنية على Node.js و Express، وواجهة أمامية (Frontend) ديناميكية باستخدام HTML5, CSS3, و JavaScript. يتم استخدام MongoDB Atlas كقاعدة بيانات سحابية.

**UI/UX Decisions:**
- **اللغة**: العربية بشكل كامل مع دعم RTL (من اليمين إلى اليسار) لجميع المكونات.
- **التصميم**: حديث وجميل باستخدام CSS3 مع تأثيرات Gradient وتصميم متجاوب (Responsive Design) ليعمل على جميع الأجهزة.
- **التدفق**: يركز على تجربة مستخدم سلسة لعمليات التسجيل، تسجيل الدخول، وإكمال الملف الشخصي.

**Technical Implementations & Feature Specifications:**
- **نظام المصادقة (Authentication)**: يعتمد على JSON Web Tokens (JWT) الصالح لمدة 7 أيام، مع نظام تسجيل دخول وخروج آمن. يتضمن نظام تحقيق ثنائي عبر البريد الإلكتروني لتفعيل الحسابات.
- **إدارة الملف الشخصي**: يوجه المستخدمين لإكمال ملفاتهم الشخصية بشكل تدريجي بعد التسجيل، مع حفظ جميع البيانات في MongoDB.
- **نماذج البيانات**:
    - **المستخدمون (الباحثون عن عمل)**: الاسم، الرقم القومي (14 رقم)، البريد الإلكتروني، رقم الهاتف، كلمة المرور، البيانات التعليمية (الجامعة، المدرسة الثانوية)، بيانات العمل (المحافظة، الراتب المطلوب، ساعات العمل)، المهارات (بحد أقصى 7)، اللغات (بحد أقصى 7)، الشهادات (اسم الشهادة ورابط PDF).
    - **الشركات**: اسم الشركة، البريد الإلكتروني، رقم الهاتف، كلمة المرور، المقر الرئيسي (المحافظة)، مجال العمل، عدد الموظفين، شعار الشركة (اختياري).
- **التحقق من البيانات**: تطبيق قواعد صارمة للتحقق من صحة المدخلات مثل الرقم القومي (14 رقم)، رقم الهاتف (11 رقم يبدأ بـ 01)، والبريد الإلكتروني، وتشفير كلمات المرور باستخدام `bcryptjs`.
- **API Endpoints**: مسارات API واضحة ومنظمة للمستخدمين (`/api/users`) والشركات (`/api/companies`) لتسجيل الدخول والتسجيل والتحقق والحصول على بيانات الملف الشخصي.

**System Design Choices:**
- **Backend Framework**: Node.js 20 و Express 5 لتوفير أداء عالٍ وقابلية للتوسع.
- **Database**: MongoDB Atlas مع Mongoose 8 لإدارة البيانات بشكل مرن وقوي.
- **Email Service**: Nodemailer مع Gmail SMTP لإرسال رسائل البريد الإلكتروني الاحترافية (أكواد التحقق) بدعم كامل للغة العربية ومعالجة محسّنة للأخطاء.
- **Security**: JWT للمصادقة، تشفير كلمات المرور، التحقق من البريد الإلكتروني، التحقق من صحة البيانات المدخلة، وحماية CORS.

## External Dependencies
- **MongoDB Atlas**: قاعدة بيانات سحابية NoSQL لتخزين بيانات المستخدمين والشركات.
- **Nodemailer**: مكتبة Node.js لإرسال رسائل البريد الإلكتروني، تستخدم Gmail SMTP.
- **JWT (jsonwebtoken)**: مكتبة لتوليد والتحقق من رموز المصادقة.
- **bcryptjs**: مكتبة لتشفير كلمات المرور.
- **CORS**: لإدارة سياسات المشاركة عبر المصادر.
- **dotenv**: لإدارة متغيرات البيئة.
